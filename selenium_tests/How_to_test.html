How To Run Our Automated Browser Tests

Welsome to the automated browser testing section of the Debate App. This page is a basic overview of how to get started with our tests. Currently, we support Chrome and Firefox.

Getting Started

To run any of these tests, you will need to have Selenium installed on your system.

  1. Python needs to be installed: Python downloads
  2. Make sure you have pip installed: Pip documentation
  3. Install Selenium using the command "pip install -U selenium" More download information here.
  4. Install the browser you are testing on your system: Chrome Firefox
  5. Our test folders include the drivers for each browser, and these specific paths are referenced in the code. However, you can download your own drivers if necessary. More information here.

You should now be able to run Selenium tests in python. If you need more help, check out the official documentation. Tests can be run from the console as ordinary python files.
